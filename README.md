# E-Learning Web Application

This is a Django-based E-learning web application designed to provide a seamless online learning experience.

## Features

- User authentication and authorization
- Course management (creation, update, deletion)
- Video lessons and course content
- Payment integration using Razorpay
- Responsive design with Bootstrap

## Technologies Used

- **Frontend:**
  - HTML
  - CSS
  - JavaScript
  - Bootstrap

- **Backend:**
  - Django

- **Payment Integration:**
  - Razorpay
